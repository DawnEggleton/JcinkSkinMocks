While you're welcome to use the more basic BBCode version of all of the elements presented here, if you're looking for more customization - or if you want to use them in your profiles, which don't accept BBCode - then you're going to need to use HTML.

Not to fear! Everything that isn't already a basic tag (such as the b tag for bolding elements!) has been made as a custom HTML tag with custom attributes such that it's as straight forward and simple as possible. With an exception of attribute values (the part between the quotation marks), which must always be lowercase (likethis), camelCase (likeThis), or UpperCamelCase (LikeThis), they can be in any casing you prefer as well. EvEn ThIS, though that seems more trouble than it's worth!

Now, for each main element, we're going to split this up a little to make it more digestable - however, if you want to find a particular tag type as quick as possible, you're going to want to search for tag: type, where your possible types are:

  • Headers
  • Basic Tags
  • Lists
  • Blockquotes
Any posting templates - including this one - in HTML code instead of BBCode will be in Posting Templates instead.

Tag: Headers These headers are the same as those presented in BBCode, which is to say there are 5 basic headers. When using the HTML versions, though, you have some alignment and color options using custom attributes.


Basic Use To use the tags without any special attributes - just as they appear in the BBCode version - you can use the plain tag. In the plain tag, you replace the '-size' part of the opening and closing tags with your chosen size. The options are:

  • Extra Large Header: '-xl'
  • Large Header: '-lg'
  • Medium Header: '-md'
  • Small Header: '-sm'
  • Extra Small Header: '-xs'
Copy to clipboard
<tag-size>Header text</tag-size>


For Example,

Copy to clipboard
<tag-xl>Extra Large Header</tag-xl>


With Offset Firstly, remember that offset only works with the three largest headers (-xl, -lg, and -md). I mention this because the tag itself is actually the same - it's just an extra attribute and a contained tag that differentiate this from the above. No lists of variables necessary! Just make sure to replace -size with your chosen size from the three options mentioned and that'll be that!

Copy to clipboard
<tag-size tag-offset>smaller text<tag-offset>offset text</tag-offset></tag-size>


For Example,

Copy to clipboard
<tag-lg tag-offset>Large Header<tag-offset>With Offset</tag-offset></tag-lg>


Change Color Want to change the font color? No big deal! Add the data-color attribute to your tag and toss a value in for it. Keep in mind that these must be lowercase, camelCase, or UpperCamelCase to work. Please note that for headers with offset, this changes the smaller background text, not the offset text. In addition, primary color and alternate color refer to white/black - primary is the default for the element and alternate is the opposite.

  • Primary Color: main
  • Alternate Color: alt
  • Bold Accent: accent
  • Mid Accent: accent2
  • Faded Accent: accent3
Copy to clipboard
<tag-size data-color="colorValue">Header Text</tag-size>


For Example,

Copy to clipboard
<tag-md data-color="accent2">Medium Header</tag-md>


Want to change the offset text color? Also not a problem! Add a value to the data-offset attribute and away you go! Keep in mind that, because this defaults to the medium accent, the values give you a somewhat different color than the above:

  • Primary Color: main
  • Alternate Color: alt
  • Main Font Color: accent
  • Darkest Accent: accent2
  • Lightest Accent: accent3
Copy to clipboard
<tag-size data-offset="offsetValue">Header Text<tag-offset>With Offset</tag-offset></tag-size>


For Example,

Copy to clipboard
<tag-md data-offset="accent3">Medium Header<tag-offset>With Offset</tag-offset></tag-md>

But what about the background on small and extra small headers, you ask? Never fear - that's an option too! For this, we're going to use the data-bg attribute and the same value options as font color. I'll repeat them here for you:

  • Primary Color: main
  • Alternate Color: alt
  • Darkest Accent: accent
  • Medium Accent: accent2
  • Lightest Accent: accent3
Copy to clipboard
<tag-size data-bg="backgroundValue">Header Text</tag-size>


For Example,

Copy to clipboard
<tag-sm data-bg="accent2">Small Header</tag-sm>

Can you combine these? You bet! You just can't combine background with offset, as offset is only available on medium to extra large headers and background is only available on extra small to small headers. Here's a couple examples:

Copy to clipboard
<tag-lg data-offset="accent" data-color="accent3">Light Accent<tag-offset>Main Color</tag-offset></tag-lg>
            <tag-xs data-bg="accent3" data-color="alt">light accent background, black text (on light mode)</tag-xs>


Change Alignment You can also align the headers to the center or right of their content space. Technically, you can also align it to the left - but it already does that by default in almost every scenario! For this, you can use data-align as your attribute, with the following values in lowercase, camelCase, or UpperCamelCase:

  • Left: left
  • Center: center
  • Right: right
Copy to clipboard
<tag-size data-align="alignValue">Header Text</tag-size>


For Example,

Copy to clipboard
<tag-xs data-align="right">Right Aligned XS Header</tag-xs>

Tag: Basic Tags There are common elements across both BBCode and HTML: bold, italic, underside, strike-through, hyperlinks, and divider lines (the hr tag). There's no real customization here that I have to offer you, beyond 'hey, try wrapping them in the group tags to see what colors you get' (only applicable for ones that are already colored though)! That said, historically, some people like to know what their options are - so here's a sample of each of those in HTML, for your convenience!

Copy to clipboard
<b>Bold</b>
            <i>Italic</i>
            <u>Underlined</u>
            <s>Strike-through</s>
            <a href="URLHERE">hyperlink</a>
            <hr>

Tag: Lists When we're asking you to give us a list of quick facts on your character, we better give you an easy way to do that - right? We thought so, anyway! And so, we have styled lists. For those who haven't used HTML lists before, there's a basic structure for them that's similar to BBCode:

Copy to clipboard
<ul>
            <li>List item</li>
            <li>List item</li>
            </ul>


Pretty straight forward, yeah? Well. Do you want a list without borders? What about a list with the plus sign enclosed in a rounded square for some flare? Maybe change the color combinations? Easy as pie! We made you some custom attributes! Read on!


Borders This one's easy - just add the attribute and set it to false. Yes, you read that right - there's only one value for this attribute right now.

Copy to clipboard
<ul data-borders="false">
            <li>List item</li>
            <li>List item</li>
            </ul>


Alternate List Marker You've seen me use it by now - that different list marker with the background behind the plus sign. How do you get that? Using the data-square attribute! The value of it is what sets the color, according to the following list:

  • Primary Color: main
  • Alternate Color: alt
  • Darkest Accent: accent
  • Medium Accent: accent2
  • Lightest Accent: accent3
Copy to clipboard
<ul data-square="squareValue">
            <li>List item</li>
            <li>List item</li>
            </ul>


For Example,

Copy to clipboard
<ul data-square="accent">
            <li>List item</li>
            <li>List item</li>
            </ul>


Change Marker Color Much like the background behind the marker, you can change the marker color itself as well using the data-bullet attribute. By default, it will be black (on light mode) or white (on dark mode). You can pick the color from the same value set as above:

  • Primary Color: main
  • Alternate Color: alt
  • Darkest Accent: accent
  • Medium Accent: accent2
  • Lightest Accent: accent3
Copy to clipboard
<ul data-bullet="squareColorValue">
            <li>List item</li>
            <li>List item</li>
            </ul>


For Example,

Copy to clipboard
<ul data-bullet="accent2">
            <li>List item</li>
            <li>List item</li>
            </ul>


You can combine all combinations of these as well - for example, the list style I've been using! See below:

Copy to clipboard
<ul data-borders="false" data-square data-bullet="accent3">
            <li>List item</li>
            <li>List item</li>
            </ul>


Wait- didn't I forget something? Actually, no! When you leave an attribute blank (like data-square, where I wanted to use the default value), you can just leave the ="" part out entirely. That's just as acceptable! The below is the exact same as the above:

Copy to clipboard
<ul data-borders="false" data-square="" data-bullet="accent3">
            <li>List item</li>
            <li>List item</li>
            </ul>


Alternate List Type Want something that looks a little different? Not a problem! We have a version with a background and no markers as well. Please keep in mind that because there are no border lines or markers, data-borders and data-bullet won't work! However, data-block can set the background color with the following values:

  • Primary Color: main
  • Alternate Color: alt
  • Darkest Accent: accent
  • Medium Accent: accent2
  • Lightest Accent: accent3
Copy to clipboard
<ul data-block="backgroundValue">
            <li>test</li>
            <li>test</li>
            </ul>


For Example,

Copy to clipboard
<ul data-block="accent3">
            <li>test</li>
            <li>test</li>
            </ul>

Tag: Blockquotes Alright, we have one more to go - blockquotes! There's the one simple style with no custom attributes:

Copy to clipboard
<blockquote>Content here</blockquote>


But you're not going to really use that much, except in your profiles, even if you like it - because, frankly, the BBCode version is identical, so why bother? But, if you want to have some fun with colors - that's when you're going to have to get into these custom attributes.


Background Color The background color can be changed using the data-block attribute. Keep in mind that, in this instance, default means black in light mode and white in dark mode. Don't want default? No big deal! There's a couple values:

  • Default: (no attribute value; leave it blank)
  • Darkest Accent: accent
  • Medium Accent: accent2
  • Lightest Accent: accent3
Copy to clipboard
<blockquote data-block="blockValue">Content here</blockquote>


For Example,

Copy to clipboard
<blockquote data-block="accent2">Content here</blockquote>


Text Color What about changing text color? You betcha! And this works for both the standard blockquote and the one with a background on it - the only difference is that the alt value does not work without the above data-block attribute because... well. The color would match the background and you can't really read that, can you? However, swapping black for white (or vice versa) on certain background colors can help with readability, so use it as you see fit! Now, for the values you have at your fingertips:

  • Primary Color: main
  • Alternate Color: alt
  • Darkest Accent: accent
  • Medium Accent: accent2
  • Lightest Accent: accent3
Copy to clipboard
<blockquote data-color="textColorValue">Content here</blockquote>


For Example,

Copy to clipboard
<blockquote data-color="accent2">Content here</blockquote>


Or, if you wanted to combine the attributes, you could get something like this:

Copy to clipboard
<blockquote data-block="accent3" data-color="alt">content here</blockquote>
